Versão 1.0 é a versão de lançamento do programa. Nesta versão estão funcionando as seguintes propriedades:

1. Cronus Add-In: O suplemento para Excel "Cronus Add-In" foi testado e já pode ser utilizado.
1.1 Funções extras: Funções extras adicionadas com o suplemento, disponiveis na aba "Fórmulas":
1.1.1 Inserir Estilos: inserir estilos do padrão visual aplicado em workbooks do programa.
1.1.2 ProcX: Função para imitar a nova ProcX disponivel nas novas versões do office.
1.1.3 User: Função para inserir nome do usuário usando atual workbook.
1.1.4 Unir Texto: Unir diversos textos com separador delimitado
1.1.5 Concatenar e somar: Concatenar e somar valores
1.1.6 Extrair LNTR: Extrair LNTR de códigos PROF
1.1.7 Código SAP: Criar código SAP para itens PROF
1.1.8 Extrair comentários: Extrair comentários de celulas em forma de texto.
1.2 Add-In Cronus™: suplemento que adiciona varias funcionalidades quando a Script SAP e Workbooks para otimizar trabalho manual.
1.2.1 Grupo SAP - Criar MM01/ZM01: Grupo para criar novos cadastros no SAP nas transações MM01 e ZM01
1.2.1.1 Atual como fonte> prematerial: Workbook atual como fonte para criar um prematerial (engloba Bobina + Tira)
1.2.1.2 Atual como fonte> PROF: Workboo atual como fonte de consulta para cadastrar um PROF (perfil, tubo normal e tubo directforming)
1.2.1.3 Atual como fonte> AO: Workbook atual como fonte de consulta para cadastrar um After Operation (tubos cortados em serra, laser, laser 3D, dobra, calandra, etc)
1.2.1.4 Atual como fonte> PURLINS: Workbook atual como fonte para cadastro de suportes para PURLINS somente.
1.2.1.5 Novo, LNTR: Criar um novo código de material LNTR (chave LNP)
1.2.2 Grupo SAP - Modificar: Modifcar cadastros e tabelas em massa.
1.2.2.1 MM02> atualizar variante: Atualizar variante de cadastros (MRP3)
1.2.2.2 MM02> Grupo de mercadorias: alterar grupo de mercadorias de cadastros
1.2.2.3 MM02> Setor de atividade: Alterar setor de atividade de cadastros em geral
1.2.2.4 MM02> NCM: Alterar NCM de cadastros em geral
1.2.2.5 MM02> Hierarquia de produto: Alterar hierarquia de produto de cadastros em geral
1.2.2.6 MM02> Cliente responsável: Alterar cliente responsável por comprar o código (Classe 001 MEINCOLMAT)
1.2.2.7 CU60> Modificar Existente> largura (LNTR): alterar a largura de bobina e tira por LNTR
1.2.2.8 CU60> Modificar Existente> prematerial (LNTR): alterar o prematerial (bobina+tira) por LNTR
1.2.2.9 CU60> Adicionar novo> Material componente: Adicionar novo componente na tabela de materiais por LNTR
1.2.3 Grupo SAP - Relatórios: gerar relatórios no SAP
1.2.3.1 Curto Create: Create an curto from ERP to APO sistem.
1.2.4 Grupo Excel - Workbooks: Workbooks comumente usados para auxilio nas tarefas.
1.2.4.1 Estrutura de Cadastros: Criar uma nova planilha para estrutura de cadastros
1.2.4.1.1 Navegar> Estrutura de produto: navegar entre as abas da planilha
1.2.4.1.2 Navegar> Estrutura de MP: navegar entre as abas da planilha
1.2.4.1.3 Navegar> Listas de Dados: navegar entre as abas da planilha
1.2.4.1.4 Inserir> Nova estrutura: inserir nova estrutura de porduto ou MP (com base na aba ativa)
1.2.4.1.5 inserir> Dados: inserir dados com base na celula ativa.
1.2.4.1.6 Cálculos> Peso Linear: calular peso linear de tubos ou perfis sem perda de scrap
1.2.4.1.7 Cálculos> Tempo de corte: Calcular tempo de corte de tubos em serra
1.2.4.1.8 Cálculos> Velocidade: Calcular velocidade com eficiencia aplicada (por máquina)
1.2.4.2 Liberação de cadastros: Enviar novos cadastros para serem liberados pelos responsaveis (reunião Outlook)
1.2.4.2.1 itens: selecionar quantos itens serão enviados. A planilha adiciona ou exlcui linhas com base no valor informado.
1.2.4.2.2 Novo Email: Enviar cadastros preenchidos na tabela para liberar. Responsaveis devem ser selecionados nesta janela.
1.2.4.3 Ficha Técnica: criar nova ficha técncia para tubos padrão.
1.2.4.4 Estudos de viabilidades: Criar novo estudo de viabilidades
1.2.4.4.1 navegação> home: ir para aba incial
1.2.4.4.2 navegação> Listas de Dados: ir para as listas de dados responsaveis pelos calculos das formulas
1.2.4.4.3 navegação> [todos demais]: Ir para abas de auxilio de calculos
1.2.4.4.4 novo estudo> Rollforming e AO: Criar novo estudo para itens de Rollforming e After Operation
1.2.4.4.5 novo estudo> Conjuntos Soldados: Criar novo estudo de Conjuntos Soldados
1.2.4.4.6 Automação> Gerar Arquivo de carga: gerar arquivo de carga com Cronus [em andamento, sem funcionamento garantido]
1.2.4.4.7 Ajuda: abrir arquivo de ajuda [em andamento, sem funcionamento garantido]
1.3 GPE Automation: Automação do GPE com Cronus ainda não é garantida e o desenvolvimento deste módulo foi pausado indefinidamente.
